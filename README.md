# 🎬 Netflix Data Science Project

## 📌 Project Overview

This project analyzes the Netflix Movies and TV Shows dataset using Python and Machine Learning.

The project includes:

- Data Cleaning
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Visualization
- Machine Learning Models
- Model Comparison

---

## 📂 Dataset

The dataset contains information about Netflix content such as:

- Type (Movie / TV Show)
- Release Year
- Rating
- Duration
- Country
- Genres
- Date Added

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 📊 Exploratory Data Analysis

The following analyses were performed:

- Missing Values Handling
- Duplicate Removal
- Outlier Detection
- Feature Encoding
- Distribution Analysis
- Correlation Analysis
- Visualizations

---

## 🤖 Machine Learning Models

Three models were trained and compared:

- Logistic Regression
- Decision Tree
- Random Forest

The target variable is:

```
Type (Movie / TV Show)
```

---

## 📈 Results

Different feature sets were tested to evaluate model performance.

The Random Forest model achieved the best performance on this dataset.
